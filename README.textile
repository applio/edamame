h1. Beanstalk + Tokyo Tyrant = "Edamame":http://en.wikipedia.org/wiki/Edamame

Edamame combines the Beanstalk priority queue with a Tokyo Tyrant database to
get a persistent, lightweight networked queuing system.







h2. Requirements

For the beanstalk part, You'll need libevent >= 1.4, beanstalkd >= 1.3, and beanstalk-client
